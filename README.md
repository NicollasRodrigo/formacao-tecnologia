## Sobre este repositório

Este repositório concentra material prático: aulas, exercícios, pequenos projetos e experimentos. A organização prioriza facilidade de navegação para revisão e demonstração de evolução técnica.

Use o repositório para:

- Consultar exemplos de código e exercícios comentados.
- Acompanhar trilhas de estudo organizadas por temas.
- Referenciar projetos compactos que mostram competências práticas.

- Fundamentos de JavaScript: variáveis, tipos, estruturas de controle

Consulte [AULAS.md](AULAS.md) para navegar por trilhas e selecionar aulas específicas.

## Como rodar e testar localmente

1. Clone o repositório:

```bash
git clone https://github.com/NicollasRodrigo/CURSO-DE-JAVASCRIPT.git
```

2. Abra o arquivo `index.html` no navegador ou use um servidor local simples (por exemplo, `Live Server` no VS Code).

3. Para executar scripts Node.js (quando houver), use o Node.js instalado localmente.
